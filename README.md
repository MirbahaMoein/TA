# TA Tools

This repo contains tools that automate my technical analysis.


## Features

- Calculate percentage of price difference and copy to clipboard


## Installation

- First of all, you need to install [python3](python.org) on your machine.
- Clone this repository on your machine
- Then you need to install requirements, so run the commands below in the CLI: 

```
cd "PATH/TO/PROGRAMS/MAIN/FOLDER"
```
```
pip install -r requirements.txt
```
